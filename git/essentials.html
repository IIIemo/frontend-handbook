# Сокращения (алиасы)

Настройте набор сокращений для часто используемых команд. Некоторые сокращения стали стандартом де-факто:

st = status
co = checkout
br = branch
ci = commit
lol = log --graph --decorate --pretty=oneline --abbrev-commit --all

Вы всегда можете добавлять/изменять/удалять сокращения. Следите за своими привычками, за часто используемыми командами.
